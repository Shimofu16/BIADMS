Barangay Information and Aid Distribution Management System

---

Installation Guide

1. Requirements

PHP 8.0 or higher

MySQL / MariaDB

Apache or Nginx (XAMPP, Laragon, or similar)

Web browser



---

2. Clone or Download the Project

git clone https://github.com/Shimofu16/BIADMS.git

Or download and extract the ZIP file.


---

3. Configure the Database

1. Create a database (e.g. biadms‎)


2. Import the provided SQL file


3. Set Up the Project

Place the project inside your web server directory:

htdocs/ (XAMPP)

www/ (Laragon)



---

5. Run the Application

Open your browser and navigate to:

http://localhost/biadms‎
